git -ssh-agent使用方法：

每次开机需要使用git，第一步从c目录ssh-agent bash开启ssh-agent程序，ssh-add +私钥，输入密码，把私钥加载进ssh-agent。第二步进入仓库所在地址，执行git操作，如果此时提示git config配置信息，切勿配置全局信息。

虾老师私信交流

```
1.制定交易模式
2.制定交易规则
3.每一笔交易按照交易规则交易，记录实绩
4.回测交易模式，看盈利的概率--[也就是我理解的雷公的盈亏比]
5.判断亏损原因，模式问题修改模式，不是模式问题，严格约束自己的交易纪律
6.反复执行前5个步骤，磨炼3-5年，等待量变引起质变，吃透模式，形成稳定盈利

注意：真正的投资永远是一个体系的事情。
有人讲选股，有人讲K线，有人讲战法，有人讲情绪，这些说到底，都只是体系的一小块拼图而已。
最简单，最简单的体系你也必须要解决三个问题。
干不干，干多少，干什么
比如昨天，优秀的短线选手看见早盘银行，保险主拉高开，明显是磨刀行情，不干。
见得多才能识得广。
先形成稳定盈利，也就是差不多一年收益率是正的，一般一年会经历2-3次小型牛熊转换，整体收益率为正，就说明你有基本的自保能力。后面再经历5-10年的大周期牛熊转换，无非就是周期长一点罢了。
一般来说经历一次大型牛熊转换，类似08,15年那种，你没被市场收割到退出股市，你才谈得上是一个成熟的交易者。
```
